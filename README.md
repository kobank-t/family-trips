# family-trips

家族旅行の旅程表を GitHub Pages で公開するリポジトリ。

🔗 https://kobank-t.github.io/family-trips/

## このリポジトリについて

- HTML は自動生成されます。直接編集しないでください
- データ管理・公開操作は [ai-sandbox](https://github.com/kobank-t/ai-sandbox) から行います

## 旅行を追加する手順

ai-sandbox で作業します。

```bash
# 1. 旅行データを作成
#    trips/{YYYY-MM-name}/data.json を作成してJSONを記述

# 2. スキルで公開
/publish-trip trips/{YYYY-MM-name}/data.json
```
